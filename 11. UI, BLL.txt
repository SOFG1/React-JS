



	Когда мы отрисовываем элементы как: сообщения, пользователи, посты и т.д. - данные не будут в наших JSX компонентах. 

	Они будут приходить с сервера, прокидываться через props-ы и отображаться.

	В react весь код делится на данные и интерфейс.

	BLL - Business Logic Layer - это код(данные) которые будут отображаться. В основном приходят с сервера.

	UI - User Interface - Код, который отрисовывает интерфейс.


	
	

	Так чтобы отобразить данные из BLL нам нужно записать каждый массив с компонентами в JSX. Т.е. если мы пишем массив в JSX то компилятор отображает все его элементы по очереди.

	Значит чтобы отобразить массив с данными мы должны применить метод .map() в JSX, который возвращает компоненты содержающие данные.

	Пример:

		props.messages = ["first message", "second message", "third message", "fourth message"];

		Пример рендеринга этих сообщений в JSX разметке:   props.messages.map( message => <Message text={message}/>);
