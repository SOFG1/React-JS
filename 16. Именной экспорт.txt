



		Именной экспорт - экспорт переменного, с использованием его названия.

	Импорт переменного с именным экспортом происходит иначе. Отличие в том что точное название при импорте мы должны писать в фигурных скобках.


	Пример:


/// Exporter ////
export let showMes = (message) {
				 alert(message);
				}
/// Exporter ////





/// Importer ////
import {showMes} from './exporter'				// Значения в фигурных скобках будет искать в имменных экспортах(не дефолт), надо указывать точное название.

/// Importer ////



	Можно менять название с помощью as пример: import {showMes as messageFunc} from './exporter'
