

	React - библиотека, которая занимается отрисовкой фронт-энд части проекта по соответствию с BLL.

	Redux - библиотека для управления BLL.


1) Запускаем  терминал : npx create-react-app 	project-name. 		PS - Путь к директории не должен содержать кирилицу или пробелы. Также точку(.) тоже лучше не использовать

2) После установки у нас есть проект. Наш проект, также является пакетом внутри Node JS - как все модули npm. Который содержит package.json.

3) Далее заходим в папку проекта и запускаем - npm start

4) После приложение запустит локальный live-server и будет открыто index.html в браузере, в котором ведется разработка нашего SPA.

5) Чтобы папка считалась проектом, необходимо наличие package.json




	1) npx create-react-app

	2) npm start
