




	Компонент в React - это функция, которая возвращает JSX разметку.		// Пример - функция App();

		Примеры: function Welcome() { return (<h1>Hello World</h1>) }		

			 let Welcome = () => { <h1>Hello World </h1>}


	PS - Если содержимое return-а  должно начаться на одной строке с return-ом или должно писаться в скобках.



	1) Название функции или переменной компонента должно начинаться с Заглавной буквы.

	2) После декларирования функции(компонента) в реакте у нас появляется новый html тег с названием функции(компонента) 	// С приведенным примером сверху у нас появился тег - <Welcome />

	3) Также мы можем вставлять компоненты друг в друга с помощью тега в app.js

	4) Чтобы использовать компоненты в других jsx файлах мы должны их импортировать. 



	PS - компоненты, которые(не универсальны) мы намерены использовать только в одном компоненте - мы можем обьявить его прямо внутри родительского компонента, без импортирования. НО не рекомендуется так делать.




	PS - При дублировании HTML разметки нам всегда нужно, подумать о занесении элементов в компоненты.
