	


		React-Redux - Библиотека, которая помогает работать со связкой React и Redux

	npm i react-redux

	React Redux - содержит инкапсулированные методы работы с Redux. Например методы работы с контекстом, store-ом, state-ом, connect... Что добавляет удобства в работе.

	Redux - библиотека, которая может использоваться и с нативным JS или с Angular. React-Redux предназначен и удобен для работы с React.

	RR - содержит компонент Provider, который будет создавать контекст для вложенных компонентов.
