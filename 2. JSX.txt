

		JSX - Разметка, которая используется в React JS. Микс JS и HTML, который выполняется в app.js.


	Отличие реакта от обычных сайтов в том, что на сайт не передается html с контентом, а почти пустой index.html со скриптами, которые динамично создают контент на клиенте. 

	Этот index.html передается 1 раз, и клиент больше не получает html, а начнет получать много скриптов, которые динамически выстроят html.

	Все это будет происходить в <div id="root"> в index.html

	Также, при запросе на сервер клиент получает не html а JSON файл, далее преобразуется в контент.




	В app.js Пишем:

	Пишем функцию, которая возвращает один html элемент с содержимым страницы.

	В конце экспортируем функцию в export default Component





	1) Название класса элемента в JSX пишется - "className" т.к. "class" является ключевым словом в JS и используется в других целях.

	2) Добавление файла стилей происходит через import, а не <link>

	3) Функция, которая экспортируется и возвращает html разметку Обязательно должна возвращать Один html элемент.
