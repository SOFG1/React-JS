



			Когда мы используем react компоненты мы вставляем целый элемент с вложенными элементами и содержимым. Но во многих случаях нам нужно использовать компоненты с разным содержимым.

	Props - аргумент функции компонента. props - всегда является обькетом, который реакт засовывает в компонент. Который может содержать все типы данных(обьекты, массивы, строки)

	Если мы не указали значения для props, то props будет являться пустым обьектом.

	Чтобы присваивать значения для props мы должны добавлять их как аттрибуты в теге компонента. 	Например: <Content text="abcabc" /> 

	Реакт автоматически добавит значение аттрибута в значения обьекта props.	// props.text - будет равняться "abcabc"

	
	
	Т.е. props - аргумент функции компонента, для использования компонентов с разными значениями.	// Например для использования компонента item с разным содержимым.

	

	Пример использования props на примере постов.

	JSX файл компонента:  		const Post = (props) => {  <div>props.text</div>  }

	Использование компонента:	<Post {text="Lorem ipsum dolor"}>









	PS - также нужно соблюдать то, что jsx функция не должна пользоваться глобальными переменными, а должна использовать то что приходит в параметры.

	Т.е. компонент должен использовать ТОЛЬКО свой props и при одинаковом props ВСЕГДА должна отображать одно и тоже, независимо от глобальных изменений.
