



		State.js - файл в котором хранинтся обьект state, в котором хранятся все данные, которые приходят с сервера(BLL), согласно котрым отрисуется UI. state.js экспортирует обьект state.

		Т.Е. state это файл который по соответствию с его данными определяется состояние приложения.

		При каждом изменении в state, UI отрисуется заново.

		


		Мы можем сделать разбивку в обьекте state на обьекты, в соответствии с существующими страницами на сайте.
