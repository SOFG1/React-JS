



	Store - ООП обьект, который содержит state и методы управления state-ом. Является интерфейсом управления state-ом.



	


	------------------


	На данном этапе разработки проекта мы перенесли обьект state в ООП обьект store, а функции для управления state-ом сделали методами store(и передали через пропс).


	Нужно быть внимательным в том что, при передаче методов store-а через пропсы при вызове контекстом методов будет являтся уже не store, а props компонента !!!

	т.е. методы являются callback-ами, они отдаются кому-то, там они вызываются, поэтому контекст уже другой.

	Для этого при передаче функций мы должны привязать им контекст store		Пример:	  <App addPost={store.addPost.bind(store)}/>
