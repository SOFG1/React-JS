

				Link Routes


		Как известно при работе <Route>-ов блоки отображаются в зависимости от url, а при кликах на ссылки браузер перезагружает страницу. 

		Но т.к. мы разрабатываем SPA нам не нужна перезагрузка страницы при каждом переходе между страницами. 

		Для этого мы используем Link Routes чтобы менять url страницы, без перезагрузки.

		Для использование этих link-ов нам также нужен npm модуль react-router-dom

		Также мы не будем использовать <a> тег вместо этого у нас будет Компонент <Link to="/Route"/> из 'react-router-dom'

		Пример:

		import {Link} from 'react-router-dom' 

		
		<Link to"/route">linkname</Link>	// Теперь у нас будет меняться маршрут в браузере и не будет обновлена страница при переходе.



		PS - Ссылку на маршрут указываем в аттрибуте "to"

		

	
			<NavLink>

	<NavLink> - используется для сыслок навигаций, также у нах автоматом прописывается класс 'active' на активной ссылке.

	Но т.к. мы используем CSS модули нам нужен модифицированный класс вместо 'active'

	Для этого в аттрибуте className используем стрелочную функцию, которая возвращает класслист в виде строки в зависимости от состояния ссылки(active).

	Пример:


	<NavLink className={({isActive}) => isActive ? `${classes.link} ${classes.active}` : classes.link} 	to="/main"> MAIN  </NavLink>


	PS - Параметр isActive в скобках мы обязательно должны взять в фигурные скобки.


						Или


	<NavLink className={NavData => NavData.isActive ? `${classes.link} ${classes.active}` : classes.link}> MAIN  </NavLink>
