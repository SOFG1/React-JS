


		Мы не можем использовать функции (в том числе компоненты) из одних js файлов в других.

		Для этого мы должны использовать Import / Export

	Для импорта пишем: import functionName from './exporting file path'		// functionName - название тега по которому будет использован в этом файле...

	Для экспорта пишем export default FucntionName					// Если мы экспортируем только 1 переменную, мы делаем export default если их много, то именной export.




	Также можем импортировать файлы не из файлов, а из модулей. Например: import React form 'react'.

	При импорте CSS мы только указываем путь к файлу. Например: import './styles.css'




	Когда мы экспортируем исользуя export default Variable, не важно как мы будем называть эту переменную в импортируемом файле, мы как бы экспортируем как безымянный, а потом можем назвать как угодно.


		Auto-Import Plugin

	Плагин Auto Import - Автоматически вставляет import Component from 'Component.js' в верхней части файла. Когда мы добавляем тег импортируемого компонента в другой компонент.
