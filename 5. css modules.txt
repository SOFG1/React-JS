



		В разработке на реакт мы пишем стили для каждого компонента в отдельном css модуль файле и импортируем его в jsx компонент.

	CSS модули должны содержать суффикс 'module', например navbar.module.css	// Это меняет поведение файла.

	В CSS модулях мы можем обращаться к классам с простыми названиями например: link, img, item. Т.к. мы пишем это в CSS модулях мы избегаем применения этих стилей к элементам с таким же классом в других компонентах.

	Эти классы будут работать только у данного компонента.
	
	Это происходит потому что, React добавит к данным названиям классов префикс с названием компонента и постфикс с идентификатором чтоб сделать название класса уникальным... Например: Navbar_logo__XlfD1

	Мы должны импортировать каждый css модуль в свой компонент, не как обычный css, а следующим образом: import classes from './Component.module.css'

	classes - это обьект который содержит данные названия классов. Ключевыми словами classes являются классы(простые), которые мы применили к элементам компонента, а знаечениями являются классы сгенерированные реактом с префиксами.

	А для наименования класса в jsx компонента пишем : <div className={classes.className}></div>	// Мы обращаемся к обьекту classes и вставляем зашифрованное значение.

	Для обьекта classes можем выбрать любое другое название при импорте.




	Для добавления нескольких классов в элемент компонента(jsx) есть 2 способа:

		1) Добавляем в строку два свойства обьекта classes и применяем в jsx - 	let classList = classes.link + ' ' + classes.active;  className={classList}

		2)  Мы можем применить их сразу с применением шаблонных строк - className={`${classes.link} ${classes.active}`}


	




	PS - целью CSS модулей является то что, мы можем применять обычные названия классов к элементам компонента, предотвращяя перемешивание стилей. CSS модули как бы применяют БЭМ автоматически. 

	PS - для глобальных стилей можем использовать app.css
